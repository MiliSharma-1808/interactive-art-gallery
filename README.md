# Interactive Art Gallery Website

A modern responsive frontend web application featuring smooth animations, parallax scrolling effects and interactive UI/UX design.

## Live Demo

🔗 https://miliartgallery.netlify.app

---

## Features

- Responsive design for all screen sizes
- Smooth animations and transitions
- Parallax scrolling effects
- Dynamic category filtering
- Interactive gallery layout
- Modern UI/UX inspired design
- Clean and organized frontend structure

---

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Flexbox
- CSS Grid

---

## Project Purpose

This project was developed to enhance frontend development skills by focusing on responsive web design, animations, interactivity and user experience.
